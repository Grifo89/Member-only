# TECHNICAL DOCUMENTATION
## Pre-requisites
* Ruby version
You will need Ruby Version 2.6.5

* System dependencies
Windows/ Mac
Rails 6.0.1

## Installation and running
Clone project and run
`bundle install --without production`
`yarn install`

 Start the rails server
`rails serve`
